class antarctic.document.XDocument(*args, **values)[source]

Bases: mongoengine.Document

A XDocument is an abstract Mongo Document, e.g. instances of this document can not be instantiated. All concrete objects such as Symbols or Strategies are children of the XDocument. Having a common parent helps to share functionality
